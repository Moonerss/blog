在R语言中，为了方便使用，可以对其进行个性化设置，优化使用体验。    

[去掉控制台的加号](去掉控制台的加号)  
[更改显示小数点位数](更改显示小数点位数)  


## 去掉控制台的加号  


```r
file.edit("~/.Rprofile")
options(continue = "  ") ## 使用两个空格代替+
```

使用效果：  


```r
a <- 1
b <- 2
a +
  b
```

```
## [1] 3
```

## 更改显示小数点位数  


```r
options(digits = 3)
```

使用效果：  


```r
a <- 3.1415926
a
```

```
## [1] 3.14
```
